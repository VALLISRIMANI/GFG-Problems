<h2><a href="https://www.geeksforgeeks.org/problems/minimum-integer--170647/1?page=3&category=Mathematical&difficulty=Basic,Easy&sortBy=submissions">Minimum Integer</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">You are given an array <strong>arr</strong>&nbsp;of size&nbsp;<strong>n</strong>. Let us denote&nbsp;<strong>s</strong>&nbsp;as the sum of all integers present in the array. Among all integers present in the array, find the minimum integer&nbsp;<strong>x</strong>&nbsp;such that <strong>s</strong><strong> ≤ n*x</strong>.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 3, arr = [1, 3, 2]
<strong>Output:</strong> 2
<strong>Explanation:</strong>
Sum of integers in the array is 6.
since 6 ≤ 3*2, therefore 2 is the answer.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 1, arr = [3]
<strong>Output: </strong>3<strong>
Explanation:
</strong>3 is the only possible answer</span></pre>
