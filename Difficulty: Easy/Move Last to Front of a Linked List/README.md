<h2><a href="https://www.geeksforgeeks.org/problems/move-last-element-to-front-of-a-linked-list/1?page=1&category=Linked%20List&difficulty=Medium,Hard&sortBy=submissions">Move Last to Front of a Linked List</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 12pt;">You are given the head of a Linked List. <p><span style="font-size: 12pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 12pt;"><strong>Input: </strong>head: 2-&gt;5-&gt;6-&gt;2-&gt;1<strong>

Explanation: </strong>In the given linked list, the last element is 1, after moving the last element to the front the linked list will be 1-&gt;2-&gt;5-&gt;6-&gt;2</span></pre>
<pre><span style="font-size: 12pt;"><strong>Input: </strong>head: 2<strong>
Output:</strong> 2<strong>
Explanation: </strong>Here 2 is the only element so, the linked list will remain the same.</span></pre>
