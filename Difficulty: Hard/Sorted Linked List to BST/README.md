<ul>
<li><span style="font-size: 18.6667px;">It is guaranteed that the linked list does not contain duplicate values.</span></li>
<li><span style="font-size: 18.6667px;">The middle element would always be the root. In case of even elements, pick the root as second mid as root.</span></li>
<li><span style="font-size: 18.6667px;">You need to return root of the tree, the preorder traversal is printed by the driver code to test your result.</span></li>
</ul>
